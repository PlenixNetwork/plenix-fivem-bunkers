# ğŸª– Plenix Bunkers

**Plenix Bunkers** is a clean and configurable FiveM script that adds **interactive bunker entrances** with a smooth door animation, optional blips, interior styling, and safety rules while inside.

Designed to be lightweight, easy to edit, and **fully configurable via `config.lua`**.

---

## ğŸ“¦ Features
- Clean, optimized client-only script
- Loads bunker IPLs automatically
- Entrance markers at all bunker locations
- Optional map blips for bunker entrances
- Door opening animation (top/bottom parts) with optional camera + sounds
- Teleport inside bunker + exit back outside (optionally to the same entrance used)
- Optional bunker interior styling (**Style A / B / C**) + interior entity sets
- Optional safety rules inside bunker:
  - Disable combat controls (shooting/aiming/weapon wheel)
  - Force unarmed
  - Optional invincibility
  - Optional melee disable
- Resource-stop cleanup (camera, door parts, blips, invincibility restore)
